# ﾋﾟｷﾞﾓﾝｺﾞアナグラム

## アナグラムって何？
同じ文字を使って異なる順序で単語やフレーズを作るやつ（たぶん）

## ﾋﾟｷﾞﾓﾝｺﾞって何？
知らない。

## これはなんのツール？
アナグラムを生成して、面白い単語を見つけるツールです。

例えば、ピギモンゴを並べ替えると
```text
- ギモゴンピ
- ギモピゴン
- ギモピンゴ
- ギモンゴピ
- ギモンピゴ
- ギンゴピモ
- ギンゴモピ
- ギンピゴモ
- ギンピモゴ
- ギンモゴピ
```

こんな感じで並べ変わった文字列が出てきますね。

おそらくこの中だと「ギンゴピモ」が優勝ですね。

## 使い方

go buildでビルドしてください。

生成されたexeファイルかそれに準ずるファイルを実行すると、並べ替えたい単語を聞いてくるので、入力してください。

アナグラムが終了したらmdファイルに結果を吐き出します。
